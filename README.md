# Infinite Scroll Gallery

A photo gallery that automatically loads more images as you scroll, powered by the Unsplash API.

## Features

- Infinite scroll — new images load automatically at the bottom
- Loading spinner while fetching
- Debounced scroll event for performance
- Lazy loading for images
- Error handling for failed requests
- DocumentFragment for optimized DOM updates

## Technologies

- HTML
- CSS Grid
- Vanilla JavaScript
- Unsplash API

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Scroll down to load more images
